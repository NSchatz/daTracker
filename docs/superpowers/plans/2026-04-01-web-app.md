# Web App Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a browser-based frontend for the location tracker with full feature parity to the Android app — live map, history, places, circle management, profile.

**Architecture:** Vanilla HTML/JS/CSS with Leaflet.js for maps. Static files embedded in the Go binary via `embed.FS`. Hash-based SPA routing. Same REST API and WebSocket as the Android app. No build step.

**Tech Stack:** HTML5, CSS3, vanilla JavaScript (ES modules), Leaflet.js 1.9.4 (vendored), Go embed.FS

- [ ] **Step 1: Create static.go**

```go
package api

import (
	"embed"
	"io/fs"
	"net/http"
	"strings"
)

//go:embed all:../../web
var webFS embed.FS

func staticFileHandler() http.Handler {
	sub, err := fs.Sub(webFS, "web")
	if err != nil {
		panic("failed to create sub filesystem: " + err.Error())
	}
	fileServer := http.FileServer(http.FS(sub))

	return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
		path := strings.TrimPrefix(r.URL.Path, "/")
		if path == "" {
			path = "index.html"
		}

		f, err := sub.Open(path)
		if err != nil {
			// SPA fallback: serve index.html for unknown paths
			r.URL.Path = "/"
			fileServer.ServeHTTP(w, r)
			return
		}
		f.Close()

		fileServer.ServeHTTP(w, r)
	})
}
```

- [ ] **Step 3: Update auth middleware for ?token= query param**

Replace the Middleware function in `server/internal/auth/middleware.go`:

```go
func (a *Auth) Middleware(next http.Handler) http.Handler {
	return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
		var tokenStr string

		header := r.Header.Get("Authorization")
		if strings.HasPrefix(header, "Bearer ") {
			tokenStr = strings.TrimPrefix(header, "Bearer ")
		}

		if tokenStr == "" {
			tokenStr = r.URL.Query().Get("token")
		}

		if tokenStr == "" {
			http.Error(w, "missing authorization", http.StatusUnauthorized)
			return
		}

		userID, err := a.ParseToken(tokenStr)
		if err != nil {
			http.Error(w, "invalid token", http.StatusUnauthorized)
			return
		}
		ctx := context.WithValue(r.Context(), contextKey{}, userID)
		next.ServeHTTP(w, r.WithContext(ctx))
	})
}
```
